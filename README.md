# 💳 SpendDNA – Your Wallet's Year-End Story

**"Spotify Wrapped for your money."**  
SpendDNA decodes your financial fingerprint by analyzing six months of transaction history using Python.

---

## 📌 Project Description

SpendDNA is a transaction analytics project built using **Python Fundamentals, NumPy, and Pandas**. It processes a messy bank transaction dataset, cleans the data, identifies vendors, categorizes spending, detects anomalies, analyzes monthly and time-based spending trends, and finally generates a personalized financial report similar to a "Spotify Wrapped" for your wallet.

---

## ✨ Features

- Transaction Parsing & Data Cleaning
- Vendor Extraction
- Smart Spending Categorization
- Financial Health Score
- Monthly Trend Analysis
- Time-of-Day Spending Analysis
- Outlier Detection using Z-Score
- Spending Archetype System
- Final SpendDNA Report

## ▶️ How to Run

1. Download the repository.
2. Place `rahul_transactions.csv` in the same folder as the notebook.
3. Open the notebook:
   ```
   SpendDNA.ipynb
   ```
4. Run all cells from top to bottom.
5. The final SpendDNA report will be printed in the notebook output.
